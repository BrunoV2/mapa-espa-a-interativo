# mapa-espa-a-interativo

<h2>
  Código que visualiza un mapa interactivo de España, permitiendo explorar comunidades y provincias mediante elementos dinámicos y navegación intuitiva.
La aplicación ofrece una experiencia visual atractiva donde el usuario puede interactuar con el mapa para consultar información geográfica de forma rápida y sencilla.
</h2>
